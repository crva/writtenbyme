# WrittenByMe Frontend

A modern React + TypeScript web application built with Vite and shadcn/ui for managing and publishing articles with analytics.

## Key Features

- **Article Management** - Create, edit, update, and delete articles with slug-based URLs
- **Real-time Analytics** - Track page views, unique visitors, and geographic data
- **Authentication** - Secure magic link authentication with session management
- **Premium Payments** - Integrated payment system for subscription upgrades
- **Modern UI Components** - Clean, accessible components from shadcn/ui
- **Type Safety** - Full TypeScript support for better development experience

## Tech Stack

- **Framework** - React 19 with TypeScript
- **Build Tool** - Vite
- **UI Library** - shadcn/ui with Tailwind CSS
- **State Management** - Zustand
- **Authentication** - Session-based with magic links

## Getting Started

```bash
npm install
npm run dev      # Start development server
npm run build    # Build for production
npm run preview  # Preview production build
```

The frontend connects to the backend API for all data operations including articles, analytics, user management, and payments.
